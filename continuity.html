2.5 Continuity

In earlier sections, we observed that many limits can be found simply by direct substitution: \[ \lim_{x \to a} f(x) = f(a). \] Functions with this property are called continuous at \(a\). Intuitively, continuity means no sudden jumps, breaks, or holes in the graph. A continuous process changes gradually, without interruption.

Definition 2.11 A function \(f\) is continuous at a number \(a\) if: \[ \lim_{x \to a} f(x) = f(a). \]

This definition requires three conditions:

  1. \(f(a)\) is defined
  2. \(\lim_{x \to a} f(x)\) exists
  3. \(\lim_{x \to a} f(x) = f(a)\)

If any of these fail, the function is discontinuous at \(a\).

Intuitively, this means that:

  • Small changes in \(x\) produce small changes in \(f(x)\)
  • No gaps, jumps, or holes
  • Graph can be drawn without lifting your pen

2.5.1 Types of Discontinuities

1. Removable Discontinuity (Hole) Occurs when:

  • Limit exists
  • Function value is missing or incorrect

Example: \[ f(x) = \frac{x^2-4}{x-2} \]

Factor: \[ f(x) = \frac{(x-2)(x+2)}{x-2} = x+2 \quad (x \ne 2) \]

Limit exists, but \(f(2)\) undefined → removable discontinuity.

2. Jump Discontinuity Left and right limits exist but are not equal.

Example: Greatest integer function \(f(x) = \lfloor x \rfloor\)
Discontinuous at every integer.

3. Infinite Discontinuity Function grows without bound.

Example: \[ f(x) = \frac{1}{x} \] Discontinuous at \(x=0\).

2.5.2 One-Sided Continuity

Definition 2.12 A function is right-continuous at \(a\) if: \[ \lim_{x \to a^+} f(x) = f(a) \]

A function is left-continuous at \(a\) if: \[ \lim_{x \to a^-} f(x) = f(a). \]

Definition 2.13 A function is continuous on an interval if it is continuous at every point in the interval (with one-sided continuity at endpoints if needed).

Theorem 2.2 If \(f\) and \(g\) are continuous at \(a\), then the following are also continuous at \(a\):

  • \(f+g\)
  • \(f-g\)
  • \(cf\)
  • \(fg\)
  • \(\frac{f}{g}\), if \(g(a) \ne 0\)

This leads to the following group of continuous functions.

Theorem 2.3 The following are continuous everywhere on their domains:

  • Polynomials
  • Rational functions
  • Root functions
  • Trigonometric functions
  • Inverse trigonometric functions
  • Exponential functions
  • Logarithmic functions

2.5.3 Composite Functions

If \(t\) is continuous at \(a\) and \(f\) is continuous at \(t(a)\), then the composite function \[ (f \circ t)(x) = f(t(x)) \] is continuous at \(a\).

Theorem 2.4 A continuous function of a continuous function is continuous.

If a function is continuous at \(a\), then: \[ \lim_{x \to a} f(x) = f(a). \] This allows direct substitution for limits.

Example 2.21 Direct Substitution

Evaluate the following limit: \[ \lim_{x \to 2} (3x^2 - 5x + 1). \]

Solution

Polynomial → continuous

\[ \lim_{x \to 2} (3x^2 - 5x + 1) = 3(2)^2 - 5(2) + 1 = 12 - 10 + 1 = 3. \]

Example 2.22 Rational Function

Evaluate the following limit: \[ \lim_{x \to 1} \frac{x^2+1}{x+2}. \]

Solution

Denominator ≠ 0 → continuous

\[ \lim_{x \to 1} \frac{x^2+1}{x+2} = \frac{1^2+1}{1+2} = \frac{2}{3}. \]

Example 2.23 Composite Function

Where is \(h\) continuous? \[ h(x) = \sin(x^2). \]

Solution

  • \(x^2\) is continuous
  • \(\sin x\) is continuous
    → \(\sin(x^2)\) is continuous everywhere

Example 2.24 Detecting Discontinuity

Where is \(f\) continuous? \[ f(x) = \ln(x) + \frac{1}{x-1}. \]

Solution

  • \(\ln(x)\): domain \(x>0\)
  • \(\frac{1}{x-1}\): undefined at \(x=1\)

So discontinuous at \(x=1\), continuous elsewhere on domain.

2.5.4 Intermediate Value Theorem (IVT)

Definition 2.14 Intermediate Value Theorem (IVT): If \(f\) is continuous on \([a,b]\) and \(N\) is any number between \(f(a)\) and \(f(b)\), then there exists \(c \in (a,b)\) such that: \[ f(c) = N. \]

Meaning: A continuous function takes every intermediate value.

Example 2.25 Root Existence

Show that the equation \[ f(x) = x^3 - 4x - 1 = 0 \] has a root in \((1,2)\).

Solution: Compute: \[ f(1) = -4 < 0, \qquad f(2) = 8 - 8 - 1 = -1 < 0 \]

Try interval: \[ f(1.5) = 3.375 - 6 - 1 = -3.625 < 0 \] \[ f(2) > 0 \]

Sign change → by IVT, at least one root exists in the interval.

2.5.5 Putting It All Together

2.5.5.1 Conceptual Takeaways

  • Continuity links limits and function values
  • Discontinuity comes from:
    • Undefined values
    • Broken limits
    • Mismatched limits and values
  • Most common functions are continuous on their domains
  • Algebra preserves continuity
  • Composition preserves continuity
  • Continuity guarantees predictable behavior
  • Continuous functions cannot “skip” values (IVT)

2.5.6 Skills You Should Be Able To Do

You should now be able to:

  • Define continuity formally
  • Identify discontinuities algebraically and graphically
  • Classify types of discontinuities
  • Use continuity to evaluate limits
  • Apply direct substitution correctly
  • Analyze domains of functions
  • Determine continuity of composite functions
  • Use algebra of continuous functions
  • Apply the Intermediate Value Theorem
  • Prove existence of roots
  • Connect graphs, limits, and function values
  • Reason about function behavior without computation
