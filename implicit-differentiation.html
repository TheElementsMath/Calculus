3.5 Implicit Differentiation

So far, most functions have been written explicitly as: \[ y = f(x) \] For example: \[ y = x^3 + 1, \quad y = x\sin x. \] In these cases, \(y\) is isolated and written directly as a function of \(x\).

Some relationships define \(x\) and \(y\) together, without solving for \(y\). For example: \[ x^2 + y^2 = 25 \] \[ x^3 + y^3 = 6xy \] These equations define curves, not single functions.

Sometimes we can solve for \(y\), but:

  • the formulas may be complicated
  • there may be multiple branches
  • it may be algebraically impractical
  • or impossible in closed form

Yet these equations still define \(y\) as a function of \(x\) locally.

Even if \(y\) is not isolated:

We can still differentiate by treating \(y\) as a function of \(x\)

So whenever we differentiate:

  • any term involving \(y\) must use the Chain Rule
  • because \(y = y(x)\)

Implicit Differentiation

Given an equation involving \(x\) and \(y\):

  1. Differentiate both sides with respect to \(x\)
  2. Treat \(y\) as \(y(x)\)
  3. Apply the Chain Rule to any \(y\)-terms
  4. Apply Product Rule and Quotient Rule when needed
  5. Collect all \(y'\) terms
  6. Solve algebraically for \(\frac{dy}{dx}\)

Example 3.28 Circle

Find \(y^{\prime}\) if \[ x^2 + y^2 = 25 \]

Solution:

Differentiate: \[\begin{align*} \frac{d}{dx}(x^2) + \frac{d}{dx}(y^2) &= \frac{d}{dx}(25)\\ 2x + 2y\frac{dy}{dx} &= 0. \end{align*}\] Solve for \(\frac{dy}{dx}\): \[\begin{align*} 2y\frac{dy}{dx} &= -2x\\ \frac{dy}{dx} &= -\frac{x}{y} \end{align*}\]

Example 3.29 Tangent Line

Find the tangent line to \[ x^2 + y^2 = 25 \] at point \((3,4)\):

Solution:

\[ \frac{dy}{dx} = -\frac{x}{y} \Rightarrow m = -\frac{3}{4} \] Point–slope form: \[ y - 4 = -\frac{3}{4}(x - 3) \]

Standard form: \[ 3x + 4y = 25. \]

Example 3.30 Folium-Type Curve

Find \(y^{\prime}\) if \[ x^3 + y^3 = 6xy \]

Solution:

Differentiate:

\[ 3x^2 + 3y^2 y' = 6x y' + 6y \]

Group \(y'\) terms: \[ 3y^2 y' - 6x y' = 6y - 3x^2 \]

Factor: \[ y'(3y^2 - 6x) = 6y - 3x^2 \]

Solve: \[ y' = \frac{6y - 3x^2}{3y^2 - 6x} = \frac{2y - x^2}{y^2 - 2x} \]

3.5.1 Horizontal Tangent Condition

Horizontal tangent means:

\[ y' = 0 \]

So numerator must be zero:

\[ 2y - x^2 = 0 \Rightarrow y = \frac{x^2}{2} \]

Substitute into original equation to find point(s).

Example 3.31 Trigonometric Implicit Function

Find \(y^{\prime}\) given: \[ \sin(x + y) = y^2 \cos x. \]

Solution:

Differentiate:

Left side: \[ \cos(x+y)(1 + y') \]

Right side (Product Rule + Chain Rule): \[ 2y y' \cos x - y^2 \sin x \]

Set equal: \[ \cos(x+y)(1+y') = 2y y'\cos x - y^2 \sin x \]

Solve for \(y'\): \[ y' = \frac{y^2 \sin x + \cos(x+y)}{2y\cos x - \cos(x+y)}. \]

Example 3.32 Second Derivatives (Implicit)

Find \(y^{\prime \prime}\) if \[ x^4 + y^4 = 16 \]

Solution:

First derivative: \[ 4x^3 + 4y^3 y' = 0 \Rightarrow y' = -\frac{x^3}{y^3} \]

Differentiate again (Quotient Rule + Chain Rule): \[ y'' = -\frac{48x^2}{y^7} \]

3.5.2 Inverse Trigonometric Functions via Implicit Differentiation

Example 3.33 Find the derivative of \(\sin^{-1}(x)\).

Solution:

Write: \(y = \sin^{-1}(x)\)

Rewrite: \[ \sin y = x \]

Differentiate: \[\begin{align*} \cos y \cdot y' &= 1\\ y' &= \frac{1}{\cos y}. \end{align*}\]

Use identity: \[ \cos y = \sqrt{1-\sin^2 y} = \sqrt{1-x^2} \]

Therefore, \[ \frac{d}{dx}(\sin^{-1} x) = \frac{1}{\sqrt{1-x^2}} \]

Example 3.34 Find \(y^{\prime}\) if \(y = \arctan(x)\).

Solution:

Write: \(y = \tan^{-1}(x)\)

Rewrite and differentiate: \[ \tan y = x \Rightarrow \sec^2 y \cdot y' = 1 \]

Therefore, \[ y' = \frac{1}{\sec^2 y} = \frac{1}{1+x^2}. \]

Theorem 3.9 \[ \frac{d}{dx}(\sin^{-1}x) = \frac{1}{\sqrt{1-x^2}} \]

\[ \frac{d}{dx}(\cos^{-1}x) = -\frac{1}{\sqrt{1-x^2}} \]

\[ \frac{d}{dx}(\tan^{-1}x) = \frac{1}{1+x^2} \]

\[ \frac{d}{dx}(\cot^{-1}x) = -\frac{1}{1+x^2} \]

\[ \frac{d}{dx}(\sec^{-1}x) = \frac{1}{|x|\sqrt{x^2-1}} \]

\[ \frac{d}{dx}(\csc^{-1}x) = -\frac{1}{|x|\sqrt{x^2-1}} \]

3.5.3 Putting It All Together

Implicit differentiation is the mathematics of hidden structure. Calculus becomes the language of structure, not just formulas.

3.5.3.1 Conceptual Takeaways

  • Not all functions can be solved explicitly
  • Many curves define \(y\) implicitly
  • Differentiation does not require solving for \(y\)
  • Variables can be dependent without isolation
  • Change propagates through relationships
  • The Chain Rule governs hidden dependencies
  • Implicit differentiation handles:
    • multi-branch curves
    • closed curves
    • self-intersecting curves
    • inverse functions
    • algebraically unsolvable equations
  • Calculus works on relationships, not just formulas
  • Geometry and algebra are unified through differentiation

3.5.3.2 Skills You Should Be Able To Do

By the end of this section, you should be able to:

  • Distinguish explicit vs implicit functions
  • Differentiate implicit equations
  • Apply the Chain Rule to \(y\)-terms
  • Solve algebraically for \(y'\)
  • Compute tangent lines from implicit equations
  • Find horizontal and vertical tangents
  • Compute second derivatives implicitly
  • Work with trigonometric implicit equations
  • Handle mixed algebraic–trigonometric forms
  • Differentiate inverse trig functions implicitly
  • Analyze curves not representable as functions
