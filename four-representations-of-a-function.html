MATH 112: Differential Calculus

1.3 Four Representations of a Function

  1. Verbal — Description in words
    Example: population depends on time

  2. Numerical — Table of values
    Example: population data by year

  3. Visual — Graph
    Example: seismograph output, population scatter plot

  4. Algebraic — Formula
    Example: \(A(r) = \pi r^2\)

A single function may be represented in multiple ways, and moving between representations improves understanding.
