2.6 Limits at Infinity; Horizontal Asymptotes

In earlier sections, we studied infinite limits where \(x \to a\) and the function values grow without bound. In this section, we instead study what happens when: \[ x \to \infty \quad \text{or} \quad x \to -\infty. \] That is, we analyze the end behavior of functions.

We investigate:

  • What happens to \(f(x)\) when \(x\) becomes very large?
  • What happens to \(f(x)\) when \(x\) becomes very negative?

Sometimes:

  • \(f(x) \to L\) (approaches a finite number)
  • \(f(x) \to \infty\) or \(f(x) \to -\infty\)
  • \(f(x)\) oscillates and does not approach anything

2.6.1 Limits at Infinity

Definition 2.15 Let \(f\) be defined for large values of \(x\). Then: \[ \lim_{x \to \infty} f(x) = L. \] means the values of \(f(x)\) can be made arbitrarily close to \(L\) by taking \(x\) sufficiently large.

Notation: \[ f(x) \to L \quad \text{as } x \to \infty \]

Definition 2.16 Let \(f\) be defined for large negative values of \(x\). Then: \[ \lim_{x \to -\infty} f(x) = L \] means the values of \(f(x)\) approach \(L\) as \(x\) becomes large in the negative direction.

Definition 2.17 The line \(y = L\) is a horizontal asymptote of \(y = f(x)\) if: \[ \lim_{x \to \infty} f(x) = L \quad \text{or} \quad \lim_{x \to -\infty} f(x) = L \]

A function may have:

  • One horizontal asymptote
  • Two horizontal asymptotes
  • No horizontal asymptotes

Example 2.26 Reciprocal Function

\[ \lim_{x \to \infty} \frac{1}{x} = 0, \qquad \lim_{x \to -\infty} \frac{1}{x} = 0 \]

So the horizontal asymptote is:

\[ y = 0 \]

Example 2.27 Polynomial Growth

\[ \lim_{x \to \infty} x^3 = \infty, \qquad \lim_{x \to -\infty} x^3 = -\infty \]

No horizontal asymptote.

Example 2.28 Exponential Function

\[ \lim_{x \to -\infty} e^x = 0, \qquad \lim_{x \to \infty} e^x = \infty \]

Horizontal asymptote:

\[ y = 0 \quad (\text{as } x \to -\infty) \]

Example 2.29 Trigonometric Oscillation

\[ \lim_{x \to \infty} \sin x \quad \text{does not exist} \]

The function oscillates forever → no limit → no asymptote.

Key Rule

For a rational function: \[ f(x) = \frac{P(x)}{Q(x)} \]

Let:

  • \(n = \deg(P)\)
  • \(m = \deg(Q)\)

Case 1: \(n < m\) \[ \lim_{x \to \pm\infty} f(x) = 0 \] Horizontal asymptote: \(y = 0\)

Case 2: \(n = m\) \[ \lim_{x \to \pm\infty} f(x) = \frac{\text{leading coeff of } P}{\text{leading coeff of } Q} \] Horizontal asymptote: ratio of leading coefficients

Case 3: \(n > m\) \[ \lim_{x \to \pm\infty} f(x) = \pm\infty \] No horizontal asymptote
(possibly slant/oblique asymptote)

Example 2.30 Evaluate the following limit:

\[ \lim_{x \to \infty} \frac{3x^2 - 2x - 2}{5x^2 + 4x + 1} \]

Solution:

Divide by \(x^2\):

\[ \frac{3 - \frac{2}{x} - \frac{2}{x^2}}{5 + \frac{4}{x} + \frac{1}{x^2}} \]

Take limits:

\[ \frac{3 - 0 - 0}{5 + 0 + 0} = \frac{3}{5} \]

Horizontal asymptote:

\[ y = \frac{3}{5} \]

A function may have two different horizontal asymptotes:

\[ \lim_{x \to \infty} f(x) = L_1, \qquad \lim_{x \to -\infty} f(x) = L_2 \]

Example:

\[ \lim_{x \to \infty} \arctan x = \frac{\pi}{2}, \qquad \lim_{x \to -\infty} \arctan x = -\frac{\pi}{2} \]

Two horizontal asymptotes: \[ y = \frac{\pi}{2}, \qquad y = -\frac{\pi}{2} \]

2.6.2 Infinite Limits at Infinity

Definition 2.18 \[ \lim_{x \to \infty} f(x) = \infty \] means the values of \(f(x)\) grow arbitrarily large as \(x \to \infty\).

Similarly: \[ \lim_{x \to -\infty} f(x) = -\infty \]

means the values of \(f(x)\) grow arbitrarily large and negative as \(x \to - \infty\).

Example 2.31 Radical Example

\[ \lim_{x \to \infty} \left(\sqrt{x^2 + 1} - x \right) \]

Solution:

Multiply by conjugate: \[ = \lim_{x \to \infty} \frac{1}{\sqrt{x^2 + 1} + x} \]

Denominator grows → expression goes to 0

\[ \lim_{x \to \infty} \left(\sqrt{x^2 + 1} - x \right) = 0. \]

2.6.3 Graph Interpretation

  • Horizontal asymptote → graph flattens toward a line
  • Infinite limit → graph grows upward/downward without bound
  • Oscillation → no limit
  • Different ends → different asymptotes

2.6.4 Putting It All Together

2.6.4.1 Conceptual Takeaways

  • Limits at infinity describe long-term behavior
  • Horizontal asymptotes describe end behavior lines
  • A function does not need to touch its asymptote
  • A function may cross a horizontal asymptote
  • Rational functions have predictable asymptotic behavior
  • Degree comparison determines asymptotes
  • Some functions approach constants (e.g., \(\frac{1}{x}, e^{-x}, \arctan x\))
  • Some grow without bound (polynomials, exponentials)
  • Some never settle (trigonometric oscillation)
  • End behavior helps with graphing and modeling

2.6.4.2 Skills You Should Be Able To Do

You should now be able to:

  • Interpret limits at infinity conceptually
  • Compute \(\lim_{x \to \infty} f(x)\) and \(\lim_{x \to -\infty} f(x)\)
  • Identify horizontal asymptotes
  • Classify infinite vs finite limits
  • Analyze rational functions using degree comparison
  • Use algebraic division by powers of \(x\)
  • Apply conjugates for radical limits
  • Recognize oscillating functions
  • Determine end behavior of polynomials
  • Connect limits to graph shape
  • Sketch graphs using asymptotes and intercepts
