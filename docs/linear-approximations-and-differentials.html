3.10 Linear Approximations and Differentials

In calculus, we often need to approximate the values of functions near a known point. One powerful method is linear approximation, based on the idea that a differentiable function looks very similar to its tangent line when zoomed in near a point.

3.10.1 Tangent Line Approximation

If a function \(f\) is differentiable at \(x = a\), the tangent line at \((a, f(a))\) can be used to approximate \(f(x)\) near \(x = a\). \[ y - f(a) = f'(a)(x - a) \] So, we can approximate \(y = f(x)\) by solving: \[ f(x) \approx f(a) + f'(a)(x - a). \]

Definition 3.3 The function \[ L(x) = f(a) + f'(a)(x - a) \] is called the linearization of \(f\) at \(a\).

Example 3.55 Linearization and Approximation

Find the linearization of \(f(x) = \sqrt{x + 3}\) at \(a = 1\) and use it to approximate \(\sqrt{3.98}\) and \(\sqrt{4.05}\).

Solution:

  1. Compute derivative: \[ f'(x) = \frac{1}{2\sqrt{x + 3}} \]

  2. Evaluate at \(a = 1\): \[ f(1) = 2, \quad f'(1) = \frac{1}{4} \]

  3. Linearization: \[ L(x) = f(1) + f'(1)(x - 1) = 2 + \frac{1}{4}(x - 1) = \frac{7}{4} + \frac{x}{4} \]

  4. Approximations: \[ \sqrt{3.98} \approx L(3.98) = 2 - 0.005 = 1.995 \]

\[ \sqrt{4.05} \approx L(4.05) = 2 + 0.0125 = 2.0125 \]

These approximations are overestimates, because the tangent line lies above the curve.

The accuracy of a linear approximation depends on how close \(x\) is to \(a\).

Example 3.56 Accuracy Interval

For \(f(x) = \sqrt{x+3}\) with linearization \(L(x) = \frac{7}{4} + \frac{x}{4}\):

  • To be accurate within \(0.5\): \(-22.6 < x < 8.6\)
  • To be accurate within \(0.1\): \(-1.1 < x < 3.9\)

Linear approximations are more accurate closer to the point of tangency.

Linear approximations simplify functions for practical use:

  • Pendulum period: For small angles, \(\sin \theta \approx \theta\)
  • Paraxial optics: \(\sin \theta \approx \theta, \cos \theta \approx 1\)

3.10.2 Differentials

Differentials provide another way to formalize linear approximations.

Definition 3.4

  • Let \(y = f(x)\). If \(dx\) is a small change in \(x\), define: \[ dy = f'(x) dx \]
  • \(dy\) approximates the actual change in \(y\) (denoted \(\Delta y\)):

Then, \[ \Delta y = f(x + \Delta x) - f(x) \]

As \(\Delta x\) becomes small, \(dy \approx \Delta y\).

Example 3.57 Using Differentials

Let \(y = x^3 + x^2 - 2x + 1\), \(x = 2\). Find \(dy\) when \(dx\) is 0.05 and 0.01.

Solution:

  • Case (a): \(dx = 0.05\) \[ dy = f'(2) \cdot 0.05 = 0.7 \] \[ \Delta y = f(2.05) - f(2) = 0.717625 \]

  • Case (b): \(dx = 0.01\) \[ dy = f'(2) \cdot 0.01 = 0.14 \] \[ \Delta y = f(2.01) - f(2) = 0.140701 \]

The approximation improves as \(\Delta x\) gets smaller.

Example 3.58 Error Estimation

A sphere has radius \(r = 21 \text{ cm}\) with a possible measurement error of \(dr = 0.05 \text{ cm}\). Estimate the maximum error in volume.

Solution:

  • Sphere volume: \(V = \frac{4}{3} \pi r^3\)
  • Differential: \(dV = 4 \pi r^2 dr\)

\[ dV = 4 \pi (21)^2 (0.05) \approx 277 \text{ cm}^3 \]

Relative error: \[ \frac{dV}{V} = 3 \frac{dr}{r} \approx 0.007. \]

The volume error is roughly three times the relative error in the radius.

3.10.3 Putting It All Together

  • Linearization approximates a differentiable function using its tangent line.
  • Differentials formalize this approximation for small changes.
  • The methods are widely used in physics and engineering to simplify computations.
  • Accuracy decreases as the point moves further from the tangent point.

3.10.3.1 Skills You Should Be Able to Do

  • Compute the derivative of a function and form its tangent line.
  • Find the linearization \(L(x)\) at a given point.
  • Use linear approximations to estimate function values.
  • Determine intervals of accuracy for linear approximations.
  • Use differentials to estimate small changes in functions.
  • Apply linear approximations in practical contexts (e.g., physics, error estimation).

3.10.3.2 Conceptual Takeaways

  • A differentiable function locally behaves like its tangent line.
  • Linear approximations give quick estimates without full calculation.
  • Differentials \(dy = f'(x) dx\) quantify small changes in the dependent variable.
  • Errors in measurement or approximation can be efficiently estimated using differentials.
  • These concepts bridge calculus with real-world applications in science and engineering.
