4.8 Newton’s Method

Many equations that arise in real-world applications cannot be solved exactly using algebraic formulas. While quadratic equations have explicit solution formulas, and cubic and quartic equations also have formulas (though complicated), there is no general formula for solving:

  • Polynomials of degree 5 or higher
  • Transcendental equations such as
    \[ \cos x = x \] or
    \[ e^x = x^2 \]

As a result, we must rely on numerical methods to find approximate solutions. One of the most important and widely used methods is Newton’s Method, also called the Newton–Raphson Method.

This method is the mathematical foundation behind how calculators and computer algebra systems solve equations numerically.

The Root-Finding Problem

We want to solve equations of the form: \[ f(x) = 0 \]

Geometrically, this means finding the x-intercepts of the graph \(y = f(x)\).

Geometric Idea of Newton’s Method

  1. Start with an initial guess \(x_1\) for the root.
  2. Draw the tangent line to the curve \(y = f(x)\) at \((x_1, f(x_1))\).
  3. Use the x-intercept of the tangent line as the next approximation \(x_2\).
  4. Repeat the process using \(x_2\), then \(x_3\), and so on.

The key idea is:

A tangent line is a good local approximation of a curve, so its x-intercept is close to the true root.

4.8.1 Derivation of the Formula

The tangent line at \(x_n\) has slope \(f'(x_n)\) and equation: \[ y - f(x_n) = f'(x_n)(x - x_n) \]

At the x-intercept, \(y = 0\), so: \[ 0 - f(x_n) = f'(x_n)(x_{n+1} - x_n) \]

Solving for \(x_{n+1}\): \[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \]

This is Newton’s Method formula.

Newton’s Method Algorithm

Given an initial guess \(x_1\): \[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \]

This produces a sequence: \[ x_1, x_2, x_3, x_4, \dots \]

If the sequence approaches a value \(r\), then: \[ \lim_{n \to \infty} x_n = r \] and \(r\) is a root of \(f(x) = 0\).

4.8.2 Convergence and Failure

Newton’s method:

  • Converges rapidly when the initial guess is good
  • Can fail when:
    • \(f'(x_n) \approx 0\)
    • The function is highly curved
    • The tangent line jumps outside the domain
    • The initial guess is poor

Thus, choosing a good starting value matters.

Example 4.35 Polynomial Equation

Solve approximately: \[ x^3 - 2x - 5 = 0 \]

Solution:

Let: \[ f(x) = x^3 - 2x - 5 \quad \Rightarrow \quad f'(x) = 3x^2 - 2 \]

Newton’s formula: \[ x_{n+1} = x_n - \frac{x_n^3 - 2x_n - 5}{3x_n^2 - 2} \]

Start with \(x_1 = 2\):

\[ x_2 \approx 2.1 \] \[ x_3 \approx 2.0946 \]

Result: \(x \approx 2.0946\)

Example 4.36 Computing a Root Value

Compute: \[ \sqrt[6]{2} \]

Solution:

This is the positive root of: \[ x^6 - 2 = 0 \]

Let: \[ f(x) = x^6 - 2 \quad \Rightarrow \quad f'(x) = 6x^5 \]

Newton’s formula: \[ x_{n+1} = x_n - \frac{x_n^6 - 2}{6x_n^5} \]

Start with \(x_1 = 1\):

\[ x_2 \approx 1.16666667 \] \[ x_3 \approx 1.12644368 \] \[ x_4 \approx 1.12249707 \] \[ x_5 \approx 1.12246205 \]

Result: \(\sqrt[6]{2} \approx 1.12246205\).

Example 4.37 Transcendental Equation

Solve: \[ \cos x = x \]

Solution:

Rewrite: \[ \cos x - x = 0 \]

Let: \[ f(x) = \cos x - x \quad \Rightarrow \quad f'(x) = -\sin x - 1 \]

Newton’s formula: \[ x_{n+1} = x_n - \frac{\cos x_n - x_n}{-\sin x_n - 1} = x_n + \frac{\cos x_n - x_n}{\sin x_n + 1} \]

Start with \(x_1 = 1\):

\[ x_2 \approx 0.75036387 \] \[ x_3 \approx 0.73911289 \] \[ x_4 \approx 0.73908513 \]

Result: \(x \approx 0.739085\).

4.8.3 Stopping Criterion

In practice, we stop when: \[ |x_{n+1} - x_n| \text{ is very small} \]

Rule of thumb:

Stop when successive approximations agree to the required number of decimal places.

4.8.4 Putting It All Together

Newton’s Method transforms calculus into a computational tool for solving problems that algebra cannot.

It is not just a technique — it is a bridge between theory, geometry, and computation.

4.8.4.1 Conceptual Takeaways

  • Many equations cannot be solved exactly
  • Numerical methods are essential in applied mathematics
  • Newton’s Method uses:
    • Tangent lines
    • Linear approximation
    • Iteration
  • It transforms geometry into computation
  • Root-finding becomes a dynamic process, not a formula
  • Calculus provides algorithms, not just symbolic results
  • Technology relies heavily on Newton’s method internally
  • Newton’s Method is an application of local linearization
  • Convergence depends on:
    • Function behavior
    • Starting value
    • Derivative size
