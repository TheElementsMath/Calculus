2.3 Calculating Limits Using the Limit Laws

In earlier sections, limits were often estimated numerically or graphically. While useful for intuition, these methods can be unreliable. This section introduces Limit Laws, which provide algebraic rules for computing limits exactly and systematically.

These laws allow us to break complicated expressions into simpler pieces and compute limits using known limits.

2.3.1 Limit Laws

Assume that
\[ \lim_{x \to a} f(x) \quad \text{and} \quad \lim_{x \to a} g(x) \] both exist, and that \(c\) is a constant.

Definition 2.7 Algebraic Limit Laws

  1. Sum Law \[ \lim_{x \to a}[f(x)+g(x)] = \lim_{x \to a} f(x) + \lim_{x \to a} g(x) \]

  2. Difference Law \[ \lim_{x \to a}[f(x)-g(x)] = \lim_{x \to a} f(x) - \lim_{x \to a} g(x) \]

  3. Constant Multiple Law \[ \lim_{x \to a}[c f(x)] = c \cdot \lim_{x \to a} f(x) \]

  4. Product Law \[ \lim_{x \to a}[f(x)g(x)] = \lim_{x \to a} f(x)\cdot \lim_{x \to a} g(x) \]

  5. Quotient Law \[ \lim_{x \to a} \frac{f(x)}{g(x)} = \frac{\lim_{x \to a} f(x)}{\lim_{x \to a} g(x)}, \quad \text{if } \lim_{x \to a} g(x) \neq 0 \]

Definition 2.8 Special Limits

  1. Constants and Identity \[ \lim_{x \to a} c = c, \qquad \lim_{x \to a} x = a \]

  2. Powers and Roots \[ \lim_{x \to a} x^n = a^n, \qquad \lim_{x \to a} \sqrt[n]{x} = \sqrt[n]{a} \]

  3. Power Law \[ \lim_{x \to a} [f(x)]^n = \left(\lim_{x \to a} f(x)\right)^n \]

Definition 2.9 Direct Substitution Property: If \(f(x)\) is a polynomial or a rational function and \(a\) is in its domain, then: \[ \lim_{x \to a} f(x) = f(a) \]

For continuous algebraic functions, limits can be found by direct substitution.

Example 2.11 Polynomial Limit

Evaluate the limit: \[ \lim_{x \to 5} (2x^2 - 3x + 4) \]

Solution

\[ = 2(5^2) - 3(5) + 4 \] \[ = 50 - 15 + 4 \] \[ = 39 \]

Example 2.12 Rational Function

Evaluate the limit: \[ \lim_{x \to -2} \frac{x^3 + 2x^2 - 1}{5 - 3x} \]

Solution

\[ = \frac{(-2)^3 + 2(-2)^2 - 1}{5 - 3(-2)} \] \[ = \frac{-8 + 8 - 1}{5 + 6} \] \[ = \frac{-1}{11} \]

Example 2.13 Indeterminate Form \(\frac{0}{0}\)

Evaluate the limit: \[ \lim_{x \to 1} \frac{x^2 - 1}{x - 1} \]

Solution

Step 1: Factor \[ = \frac{(x-1)(x+1)}{x-1} \]

Step 2: Cancel \[ = x+1 \]

Step 3: Take limit \[ \lim_{x \to 1} (x+1) = 2 \]

Example 2.14 Difference of Squares Structure

Evaluate the limit: \[ \lim_{h \to 0} \frac{(3+h)^2 - 9}{h} \]

Solution

1. Expand numerator \[ = \frac{9 + 6h + h^2 - 9}{h} \] \[ = \frac{6h + h^2}{h} \]

2. Simplify \[ = 6 + h \]

3. Limit \[ \lim_{h \to 0} (6+h) = 6 \]

Example 2.15 Rationalization

Evaluate the limit: \[ \lim_{t \to 0} \frac{\sqrt{t+9} - 3}{t} \]

Solution

1. Multiply by conjugate: \[ \frac{\sqrt{t+9} - 3}{t} \cdot \frac{\sqrt{t+9} + 3}{\sqrt{t+9} + 3} \]

\[ = \frac{(t+9)-9}{t(\sqrt{t+9}+3)} \]

2. Cancel \(t\): \[ = \frac{1}{\sqrt{t+9}+3} \]

3. Take limit: \[ = \frac{1}{3+3} = \frac{1}{6} \]

Example 2.16 Absolute Value

Evaluate the limit: \[ \lim_{x \to 0} |x| \]

Solution

1. Right-hand: \[ \lim_{x \to 0^+} x = 0 \]

2. Left-hand: \[ \lim_{x \to 0^-} (-x) = 0 \]

Therefore: \[ \lim_{x \to 0} |x| = 0 \]

Example 2.17 Nonexistent Limit

Evaluate the limit: \[ \lim_{x \to 0} \frac{|x|}{x} \]

Solution

1. Right-hand: \[ \lim_{x \to 0^+} \frac{x}{x} = 1 \]

2. Left-hand: \[ \lim_{x \to 0^-} \frac{-x}{x} = -1 \]

Since they differ: \[ \text{The limit does not exist.} \]

2.3.2 The Squeeze Theorem (Sandwich Theorem)

The Squeeze Theorem (also called the Sandwich Theorem) is a method for evaluating limits of functions that are trapped between two simpler functions whose limits are known and equal.

Definition 2.10 Theorem (Squeeze Theorem)

Suppose that for all \(x\) near \(a\) (except possibly at \(a\)), \[ g(x) \le f(x) \le h(x) \] and \[ \lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L. \] Then, \[ \lim_{x \to a} f(x) = L. \]

If a function is always between two other functions, and both of those functions approach the same value, then the trapped function has no choice but to approach that same value.

The outer functions “squeeze” the middle function into a single limit value.

Visually, think of three curves:

  • Top curve: \(h(x)\)
  • Middle curve: \(f(x)\)
  • Bottom curve: \(g(x)\)

As \(x \to a\), both \(g(x)\) and \(h(x)\) approach the same height \(L\). Since \(f(x)\) is stuck between them, it must also approach \(L\).

Example 2.18 Oscillating Function

Evaluate the limit: \[ \lim_{x \to 0} x^2 \sin\left(\frac{1}{x}\right) \]

Solution

Step 1: Bound the sine function \[ -1 \le \sin\left(\frac{1}{x}\right) \le 1 \]

Step 2: Multiply by \(x^2\) \[ - x^2 \le x^2 \sin\left(\frac{1}{x}\right) \le x^2 \]

Step 3: Take limits \[ \lim_{x \to 0} (-x^2) = 0, \qquad \lim_{x \to 0} x^2 = 0 \]

Conclusion (Squeeze Theorem): \[ \lim_{x \to 0} x^2 \sin\left(\frac{1}{x}\right) = 0 \]

Example 2.19 Absolute Value Oscillation

Evaluate the limit: \[ \lim_{x \to 0} x \cos\left(\frac{1}{x}\right) \]

Solution

Bounding: \[ -1 \le \cos\left(\frac{1}{x}\right) \le 1 \]

Multiply by \(x\): \[ - |x| \le x\cos\left(\frac{1}{x}\right) \le |x| \]

Limits: \[ \lim_{x \to 0} (-|x|) = 0, \qquad \lim_{x \to 0} |x| = 0 \]

Conclusion: \[ \lim_{x \to 0} x\cos\left(\frac{1}{x}\right) = 0 \]

Example 2.20 Fractional Squeeze

Evaluate the limit:

\[ \lim_{x \to 0} \frac{x^2}{\sqrt{x^2 + 1}} \]

Solution

\[ \sqrt{x^2 + 1} \ge 1 \]

So: \[ 0 \le \frac{x^2}{\sqrt{x^2+1}} \le x^2 \]

Taking limits: \[ \lim_{x \to 0} 0 = 0, \qquad \lim_{x \to 0} x^2 = 0 \]

Therefore, \[ \lim_{x \to 0} \frac{x^2}{\sqrt{x^2 + 1}} = 0 \]

2.3.3 Putting It All Together

2.3.3.1 Conceptual Takeaways

  • Limits behave algebraically like numbers.
  • Complicated limits can be broken into simpler components.
  • Not all limits can be evaluated by substitution.
  • Indeterminate forms like \(\frac{0}{0}\) require algebraic simplification.
  • The value of a function at a point does not affect the limit.
  • Some limits require:
    • One-sided analysis
    • Algebraic restructuring
    • Bounding arguments (Squeeze Theorem)

2.3.3.2 Skills Checklist

You should now be able to:

  • Apply all five limit laws
  • Use direct substitution
  • Factor and simplify expressions
  • Rationalize expressions
  • Identify indeterminate forms
  • Compute one-sided limits
  • Detect nonexistent limits
  • Apply the Squeeze Theorem
  • Distinguish between function value and limit value
