5.3 Example 3: Simple Harmonic Motion (Modeling Application)

Position function:

\[ s(t) = 4\cos t \]

Velocity:

\[ v(t) = -4\sin t \]

Acceleration:

\[ a(t) = -4\cos t \]

5.3.1 Interpretation:

  • Motion is periodic
  • Speed is greatest at equilibrium
  • Acceleration is greatest at endpoints
  • The system oscillates between \(s = \pm 4\)
  • Period = \(2\pi\)

This models simple harmonic motion, common in: - springs
- vibrations
- waves
- circuits
- signal processing
- mechanical systems
