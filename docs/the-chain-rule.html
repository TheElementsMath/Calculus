3.4 The Chain Rule

Many functions in calculus are composite functions — functions built from other functions.

Example: \[ F(x) = \sqrt{x^2 + 1} \]

This is not a basic function, but a composition:

  • Inner function: \(u = x^2 + 1\)
  • Outer function: \(y = \sqrt{u}\)

Previously learned differentiation rules are not sufficient for this structure alone. We need a new rule to differentiate functions inside functions.

Conceptually, if:

  • \(u\) changes with respect to \(x\)
  • \(y\) changes with respect to \(u\)

Then \(y\) changes with respect to \(x\) through both changes. So the total rate of change is: \[ \text{(change of y wrt u)} \times \text{(change of u wrt x)}. \] This leads to the Chain Rule.

Theorem 3.7 The Chain Rule

If:

  • \(u = t(x)\)
  • \(y = f(u)\)

Then the composite function: \[ y = f(t(x)) \] has derivative: \[ \frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}. \]

In prime notation, \[ (f \circ t)'(x) = f'(t(x)) \cdot t'(x). \]

Differentiate the outer function
Keep the inner function unchanged
Multiply by the derivative of the inner function

Structure:

\[ \frac{d}{dx}[f(t(x))] = \underbrace{f'(t(x))}_{\text{outer derivative}} \cdot \underbrace{t'(x)}_{\text{inner derivative}} \]

Example 3.22 Square Root Composition

Differentiate: \[ F(x) = \sqrt{x^2 + 1} \]

Solution:

Let:

  • \(u = x^2 + 1\)
  • \(y = \sqrt{u} = u^{1/2}\)

Derivatives: \[ \frac{dy}{du} = \frac{1}{2}u^{-1/2}, \quad \frac{du}{dx} = 2x. \] Apply Chain Rule: \[ F'(x) = \frac{1}{2}(x^2+1)^{-1/2} \cdot 2x = \frac{x}{\sqrt{x^2+1}}. \]

Example 3.23 Trigonometric Composition

Differentiate \(y = \sin(x^2)\) and \(y = \sin^2 x\)

Solution: \(y = \sin(x^2)\)

Outer: \(\sin(u)\)
Inner: \(u = x^2\)

So, \[ y' = \cos(x^2)\cdot 2x = 2x\cos(x^2). \]

Solution: \(y = \sin^2 x\)

Rewrite: \[ y = (\sin x)^2. \]

Outer: \(u^2\)
Inner: \(u = \sin x\)

\[ y' = 2\sin x \cdot \cos x = \sin(2x). \]

Theorem 3.8 Power Rule + Chain Rule

If \(y = [t(x)]^n\), then: \[ \frac{d}{dx}[t(x)]^n = n[t(x)]^{n-1} \cdot t'(x). \]

Example 3.24 If \[ y = (x^3 - 1)^{100}, \] then \[ y' = 100(x^3-1)^{99} \cdot 3x^2 = 300x^2(x^3-1)^{99}. \]

Example 3.25 Differentiate \[ f(x) = \frac{1}{\sqrt[3]{x^2+x+1}}. \]

Solution:

Rewrite: \[ f(x) = (x^2+x+1)^{-1/3}. \] Then, \[ f'(x) = -\frac{1}{3}(x^2+x+1)^{-4/3}(2x+1) = -\frac{2x+1}{3(x^2+x+1)^{4/3}}. \]

Example 3.26 Differentiate \[ y = e^{\sin x} \]

Solution:

Outer: \(e^u\)
Inner: \(u = \sin x\)

Then, \[ y' = e^{\sin x} \cdot \cos x = e^{\sin x}\cos x. \]

In general, \[ \frac{d}{dx}(e^{u}) = e^{u}\cdot \frac{du}{dx}. \]

Example 3.27 Multi-Layer Chain Rule

Differentiate \[ f(x) = \sin(\cos(\tan x)) \]

Solution:

Layers:

  • Outer: \(\sin(\cdot)\)
  • Middle: \(\cos(\cdot)\)
  • Inner: \(\tan x\)

Differentiate step-by-step:

\[ f'(x) = \cos(\cos(\tan x)) \cdot (-\sin(\tan x)) \cdot \sec^2 x = -\cos(\cos(\tan x))\sin(\tan x)\sec^2 x. \]

The Chain Rule reflects information flow: \[ x \rightarrow u(x) \rightarrow y(u). \] Change propagates through layers.

3.4.1 Putting It All Together

The Chain Rule is the rule of dependency.

It formalizes how change flows through systems. Calculus becomes the mathematics of structure and flow.

3.4.1.1 Conceptual Takeaways

  • Most real-world functions are composite
  • Differentiation follows structure
  • Rates of change multiply through layers
  • Inner change amplifies outer change
  • The Chain Rule is universal
  • It connects structure and motion
  • It unifies algebraic, trig, and exponential differentiation
  • It explains nested system behavior
  • It models cascaded processes
  • It formalizes dependency chains

3.4.1.2 Skills You Should Be Able To Do

By the end of this section, you should be able to:

  • Identify composite functions
  • Decompose functions into layers
  • Apply the Chain Rule correctly
  • Use Leibniz notation meaningfully
  • Combine Chain Rule with:
    • Power Rule
    • Product Rule
    • Quotient Rule
  • Differentiate nested trig functions
  • Differentiate nested exponential functions
  • Handle multi-layer compositions
  • Recognize inner vs outer functions
  • Apply structure-first differentiation
  • Work from outside to inside
  • Interpret derivatives structurally
  • Model multi-stage systems
  • Solve complex composite derivatives
  • Translate symbolic structure into derivatives
