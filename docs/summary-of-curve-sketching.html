4.5 Summary of Curve Sketching

In calculus, graphing a function is not simply about producing a visual picture — it is about understanding the structure and behavior of the function. By combining algebraic properties with calculus tools, we can reveal important features of a curve such as where it increases or decreases, where it bends, where it reaches extreme values, and how it behaves near boundaries and infinity.

Modern graphing technology can produce accurate plots, but without mathematical analysis these graphs can be misleading or incomplete. Calculus allows us to:

  • Detect subtle local maxima and minima
  • Identify inflection points
  • Determine exact locations of key features
  • Understand long-term behavior
  • Avoid false conclusions from visual approximations

A complete graph is the result of combining:

  • Algebraic structure
  • Limits
  • Derivatives
  • Second derivatives
  • Symmetry
  • Asymptotic behavior

This integration of ideas transforms graphing from plotting points into mathematical reasoning.

4.5.1 A Structured Framework for Sketching Curves

To sketch a function \(y = f(x)\) effectively, we follow a systematic analysis:

1. Domain Determine all values of \(x\) for which the function is defined.

2. Intercepts

  • \(y\)-intercept: \(f(0)\), if defined
  • \(x\)-intercepts: Solve \(f(x) = 0\)

3. Symmetry

  • Even function:
    \[ f(-x) = f(x) \Rightarrow \text{symmetry about the } y\text{-axis} \]

  • Odd function:
    \[ f(-x) = -f(x) \Rightarrow \text{symmetry about the origin} \]

  • Periodic function:
    \[ f(x+p) = f(x) \Rightarrow \text{repeating behavior with period } p \]

4. Asymptotes

Horizontal asymptotes: \[ \lim_{x \to \pm \infty} f(x) = L \Rightarrow y = L \]

Vertical asymptotes: \[ \lim_{x \to a^\pm} f(x) = \pm \infty \Rightarrow x = a \]

Slant asymptotes: \[ \lim_{x \to \pm \infty} \left(f(x) - (mx+b)\right) = 0 \Rightarrow y = mx + b \]

5. Increasing and Decreasing Behavior

Compute \(f'(x)\):

  • \(f'(x) > 0 \Rightarrow f\) increasing
  • \(f'(x) < 0 \Rightarrow f\) decreasing

6. Local Extrema

Find critical numbers where: \[ f'(x) = 0 \quad \text{or} \quad f'(x) \text{ does not exist} \]

Use:

  • First Derivative Test
  • Second Derivative Test

7. Concavity and Inflection Points

Compute \(f''(x)\):

  • \(f''(x) > 0 \Rightarrow\) concave up
  • \(f''(x) < 0 \Rightarrow\) concave down

Inflection points occur where concavity changes.

8. Final Sketch

Combine:

  • Intercepts
  • Asymptotes
  • Increasing/decreasing intervals
  • Local extrema
  • Concavity
  • Inflection points
  • Symmetry

to produce a mathematically accurate graph.

Example 4.27 Rational Function

Sketch: \[ f(x) = \frac{2x^2}{x^2 - 1} \]

Solution:

Key Features

  • Domain: \(x \neq \pm 1\)
  • Intercepts: \((0,0)\)
  • Symmetry: Even function → symmetry about \(y\)-axis
  • Asymptotes:
    • Vertical: \(x = \pm 1\)
    • Horizontal: \(y = 2\)
  • Monotonicity: Increasing on \((-\infty,-1) \cup (-1,0)\), decreasing on \((0,1) \cup (1,\infty)\)
  • Extrema: Local maximum at \(x=0\)
  • Concavity:
    • Concave up on \(|x|>1\)
    • Concave down on \(|x|<1\)

Example 4.28 Exponential Growth/Decay Interaction

Sketch \[ f(x) = x e^x \]

Solution:

Key Features

  • Domain: \(\mathbb{R}\)
  • Intercept: \((0,0)\)
  • Asymptote: Horizontal asymptote \(y=0\) as \(x \to -\infty\)
  • Critical point: \(x=-1\)
  • Minimum value: \(f(-1) = -e^{-1}\)
  • Concavity change: Inflection point at \(x=-2\)

This function shows how exponential decay dominates polynomial growth for large negative \(x\).

Example 4.29 Logarithmic Function

Sketch \[ f(x) = \ln(4 - x^2) \]

Solution:

Key Features

  • Domain: \((-2,2)\)
  • Vertical asymptotes: \(x=\pm 2\)
  • Symmetry: Even
  • Maximum: at \(x=0\)
  • Concavity: Always concave down

This graph demonstrates domain-restricted behavior and vertical asymptotic boundaries.

4.5.2 Putting It All Together

A graph is not an image — it is a mathematical story told by derivatives, limits, and structure.

Calculus gives us the language to read that story precisely.

4.5.2.1 Conceptual Takeaways

  • Graphing is not plotting — it is analysis.
  • Every curve has structure that can be revealed using calculus.
  • Derivatives describe motion and direction.
  • Second derivatives describe bending and shape.
  • Limits describe boundaries and long-term behavior.
  • Algebra reveals symmetry and structure.
  • Calculus connects local behavior (tangents) to global shape (entire graph).

A correct graph is a logical consequence of mathematical properties, not a visual guess.

4.5.2.2 Skills You Should Be Able to Do

After mastering this section, you should be able to:

  • Determine the domain of a function
  • Identify symmetry (even, odd, periodic)
  • Find intercepts
  • Compute and interpret limits
  • Identify asymptotes (horizontal, vertical, slant)
  • Compute derivatives and determine monotonicity
  • Find and classify critical points
  • Analyze concavity using second derivatives
  • Identify inflection points
  • Construct a full graph using mathematical reasoning
  • Explain why a graph behaves as it does
