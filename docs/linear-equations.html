9.5 Linear Equations

First-order linear differential equations form one of the most important classes of differential equations in calculus. They appear naturally in physics, biology, economics, engineering, and chemistry, especially in models involving growth, decay, circuits, mixing, and population dynamics.

Definition 9.4 A first-order linear differential equation is any equation that can be written in the form \[ \frac{dy}{dx} + P(x)y = Q(x), \] where:

  • \(P(x)\) and \(Q(x)\) are continuous functions on an interval,
  • \(y\) is the unknown function,
  • the equation is linear in \(y\) (no powers of \(y\), no products like \(y^2\), \(y\sin y\), etc.).

The equation is called linear because:

  • \(y\) appears only to the first power
  • \(y\) is not inside nonlinear functions
  • coefficients depend only on \(x\)

This makes the structure predictable and solvable using a general method.

Many real systems naturally produce linear equations:

  • population harvesting models
  • electric circuits
  • cooling/heating laws
  • mixing problems
  • growth with forcing terms
  • economic adjustment models

They are often not separable, but still solvable using a systematic technique.

9.5.1 The Integrating Factor Method

To solve \[ \frac{dy}{dx} + P(x)y = Q(x), \] we use an integrating factor \(I(x)\), defined by: \[ I(x) = e^{\int P(x)\,dx} \]

Multiplying the equation by \(I(x)\) transforms the left-hand side into the derivative of a product:

\[ I(x)\frac{dy}{dx} + I(x)P(x)y = \frac{d}{dx}\big(I(x)y\big) \]

So the equation becomes:

\[ \frac{d}{dx}\big(I(x)y\big) = I(x)Q(x) \]

If we integrate both sides:

\[ I(x)y = \int I(x)Q(x)\,dx + C \]

and solve for \(y\):

\[ y(x) = \frac{1}{I(x)}\left(\int I(x)Q(x)\,dx + C\right) \]

To solve a first-order linear DE:

  1. Put in standard form: \[ \frac{dy}{dx} + P(x)y = Q(x) \]

  2. Compute the integrating factor: \[ I(x) = e^{\int P(x)\,dx} \]

  3. Multiply the entire equation by \(I(x)\)

  4. Rewrite left side as a product derivative: \[ \frac{d}{dx}(I(x)y) \]

  5. Integrate both sides

  6. Solve for \(y(x)\)

Example 9.13 Basic Linear Equation

Solve \[ \frac{dy}{dx} + 3x^2 y = 6x^2 \]

Solution

Standard form is already given:

  • \(P(x) = 3x^2\)
  • \(Q(x) = 6x^2\)

Integrating factor: \[ I(x) = e^{\int 3x^2 dx} = e^{x^3} \]

Multiply through: \[ e^{x^3}\frac{dy}{dx} + 3x^2 e^{x^3} y = 6x^2 e^{x^3} \]

Left-hand side becomes: \[ \frac{d}{dx}(e^{x^3}y) = 6x^2 e^{x^3} \]

Integrate: \[ e^{x^3}y = \int 6x^2 e^{x^3} dx \]

Let \(u = x^3\), \(du = 3x^2 dx\), so: \[ e^{x^3}y = 2e^{x^3} + C \]

Solve: \[ y = 2 + Ce^{-x^3} \]

Example 9.14 Initial Value Problem

Solve \[ x^2 y' + xy = 1, \qquad y(1) = 2, \quad x>0 \]

Solution

Divide by \(x^2\): \[ y' + \frac{1}{x}y = \frac{1}{x^2} \]

Integrating factor: \[ I(x) = e^{\int \frac{1}{x}dx} = e^{\ln x} = x \]

Multiply through: \[ x y' + y = \frac{1}{x} \]

\[ \frac{d}{dx}(xy) = \frac{1}{x} \]

Integrate: \[ xy = \ln x + C \]

Solve: \[ y = \frac{\ln x + C}{x} \]

Apply initial condition \(y(1)=2\): \[ 2 = \frac{0 + C}{1} \Rightarrow C = 2 \]

Therefore, \[ y = \frac{\ln x + 2}{x} \]

9.5.2 Application Example: Electric Circuits (RL Circuit)

A simple electric circuit with:

  • resistance \(R\),
  • inductance \(L\),
  • voltage source \(E(t)\)

satisfies: \[ L\frac{dI}{dt} + RI = E(t) \] This is a first-order linear differential equation in \(I(t)\).

Solution structure: \[ I(t) = \frac{E}{R}\big(1 - e^{-Rt/L}\big) \]

This shows:

  • current starts at 0
  • increases smoothly
  • approaches a limiting value \(\frac{E}{R}\)

9.5.3 Putting It All Together

First-order linear differential equations form a powerful modeling framework:

  • Their structure guarantees solvability.
  • The integrating factor transforms structure into exact derivatives.
  • Solutions naturally separate into:
    • transient parts (decaying exponentials)
    • steady-state parts (long-term behavior)
  • Long-term system behavior is often independent of initial conditions.
  • They unify many applied models into one mathematical form.

This reveals a deep idea in calculus:

Structure determines solvability, and assumptions determine behavior.

9.5.3.1 Conceptual Takeaways

  • Linearity refers to structure, not graph shape
  • Integrating factors convert equations into exact derivatives
  • Exponential terms control stability and decay
  • Long-term behavior dominates system dynamics
  • Transient behavior fades over time
  • Steady-state solutions govern equilibrium
  • Mathematical form encodes physical laws
  • One method solves many real-world systems
