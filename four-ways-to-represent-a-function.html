MATH 112/ 122: Differential and Integral Calculus

1.1 Four Ways to Represent a Function

A function is a rule that assigns exactly one output to each input from a specified set. Functions model relationships where one quantity depends on another and appear throughout mathematics, science, and real-world applications.

Definition 1.1 A function is a rule assigning each input \(x\) in a domain \(D\) exactly one output \(f(x)\) in a set \(E\).

Definition 1.2 For any function, the domain is the set of all allowable inputs. The range is the set of all possible outputs.

Definition 1.3 For a single variable function, the independent variable is the input variable while the dependent variable is the output variable.

Conceptual Models of a Function

  • Machine Model:
    Input → Function rule → Output

  • Arrow Diagram:
    Domain elements mapped to range elements

  • Graph Model:
    Set of ordered pairs \((x, f(x))\)

Example 1.1 One Function, Multiple Interpretations

Let the function be: \[ f(x) = 2x + 1 \]

We interpret this function in multiple connected ways.

1. Independent and Dependent Variables

  • The independent variable is \(x\) (the input)
  • The dependent variable is \(f(x)\) (the output)

This means the value of \(f(x)\) depends on the value of \(x\).

2. Domain and Range

If no restrictions are given:

  • Domain: all real numbers
    \[ (-\infty, \infty) \]

  • Range: all real numbers
    \[ (-\infty, \infty) \]

So every real input produces exactly one real output.

3. Machine Model (Process View)

Think of the function as a machine:

Input (x) → multiply by 2 → add 1 → Output (f(x))

\[x = 3 → (2·3) + 1 → 7\]

So, \[f(3) = 7.\]

4. Arrow Diagram (Mapping View)

Domain → Range

\[\begin{align*} -2& → -3 \\ -1& → -1 \\ 0& → 1 \\ 1& → 3 \\ 2& → 5 \\ \end{align*}\]

Each input has exactly one arrow pointing to one output.

5. Graph Model (Geometric View)

The graph of \[f(x)=2x+1\] is a straight line:

  • slope = 2
  • \(y\)-intercept = 1

Each \(x\)-value corresponds to exactly one \(y\)-value, so the graph passes the Vertical Line Test.

Graphical Interpretation

  • The graph shows the behavior of a function.
  • Function values are read as heights above the \(x\)-axis.
  • Domain is visualized on the \(x\)-axis.
  • Range is visualized on the \(y\)-axis.

Domain Rules

  • Square roots → radicand \(\ge 0\)
  • Denominators → cannot be 0
  • Domain convention: If not stated, the domain is all real numbers where the formula is defined

Example 1.2 Identifying Domain and Range

Let \(f(x) = \sqrt{x - 3}\).

  1. Find the domain of \(f\)
  2. Find the range of \(f\)

Solution

The square root function is defined only when the radicand is nonnegative:

\[ x - 3 \ge 0 \Rightarrow x \ge 3 \]

So the domain is: \[ [3, \infty) \]

Since \(\sqrt{x-3} \ge 0\) for all \(x \ge 3\), the smallest value of \(f(x)\) occurs at \(x=3\):

\[ f(3) = 0 \]

Thus the range is: \[ [0, \infty) \]

1.1.1 Four Representations of a Function

Four Representations of a Function

  1. Verbal — Description in words
    Example: population depends on time

  2. Numerical — Table of values
    Example: population data by year

  3. Visual — Graph
    Example: seismograph output, population scatter plot

  4. Algebraic — Formula
    Example: \(A(r) = \pi r^2\)

A single function may be represented in multiple ways, and moving between representations improves understanding.

Example 1.3 Representations of a Function

The area \(A\) of a circle depends on the radius \(r\).

  1. Give a verbal representation
  2. Give an algebraic representation
  3. Describe a graphical representation
  4. Describe a numerical representation

Solution

  1. Verbal:
    The area of a circle increases as the radius increases.

  2. Algebraic:
    \[ A(r) = \pi r^2 \]

  3. Graphical:
    The graph is the right half of an upward-opening parabola in the \((r, A)\)-plane, defined for \(r \ge 0\).

  4. Numerical:
    A table of values: \[ \begin{array}{c|c} r & A \\ \hline 1 & \pi \\ 2 & 4\pi \\ 3 & 9\pi \end{array} \]

Definition 1.4 Vertical Line Test

A graph represents a function if and only if no vertical line intersects the graph more than once.

Example 1.4 Vertical Line Test

Determine whether the graph defined by the equation
\[ x = y^2 - 1 \] represents a function of \(x\).

Solution

Rewriting: \[ y^2 = x + 1 \Rightarrow y = \pm \sqrt{x+1} \]

For many values of \(x\), there are two corresponding values of \(y\).

Therefore, vertical lines intersect the graph more than once.

\[ \text{The graph does NOT represent a function of } x. \]

This fails the Vertical Line Test.

1.1.2 Piecewise-Defined Functions

Some functions are defined by different formulas on different intervals:

\[ f(x) = \begin{cases} \text{rule 1} & \text{if condition 1} \\ \text{rule 2} & \text{if condition 2} \end{cases}. \]

Examples: - Absolute value function
- Step functions (e.g., postage cost)
- Physical systems with phase changes

Example 1.5 Piecewise Function Evaluation

Let \[ f(x) = \begin{cases} x+2, & x < 0 \\ x^2, & x \ge 0 \end{cases} \]

Compute: 1. \(f(-3)\)
2. \(f(0)\)
3. \(f(2)\)

Solution

Since the function is piecewise, choose the rule based on the value of \(x\):

  • \(x=-3 < 0 \Rightarrow f(-3) = -3 + 2 = -1\)
  • \(x=0 \ge 0 \Rightarrow f(0) = 0^2 = 0\)
  • \(x=2 \ge 0 \Rightarrow f(2) = 2^2 = 4\)

So: \[ f(-3)=-1,\quad f(0)=0,\quad f(2)=4 \]

Example 1.6 Function as a Model

Suppose the cost \(C\) of shipping a package is:

  • $5 for up to 1 kg
  • $3 for each additional kg (or part thereof)

Define \(C(w)\) as a piecewise function.

Solution

\[ C(w) = \begin{cases} 5, & 0 < w \le 1 \\ 5 + 3(w-1), & w > 1 \end{cases} \]

This defines a step-type piecewise function modeling real-world cost rules.

1.1.3 Symmetry

Definition 1.5

  • Even function:
    \[ f(-x) = f(x) \] → symmetry about the y-axis

  • Odd function:
    \[ f(-x) = -f(x) \] → symmetry about the origin

Example 1.7 Even, Odd, or Neither

Determine whether the function
\[ f(x) = x^4 - x^2 \] is even, odd, or neither.

Solution

Compute \(f(-x)\):

\[ f(-x) = (-x)^4 - (-x)^2 = x^4 - x^2 = f(x) \]

Since: \[ f(-x) = f(x) \]

The function is even.

Therefore, its graph is symmetric about the y-axis.

1.1.4 Increasing and Decreasing Functions

Definition 1.6

  • Increasing on interval \(I\): \[ x_1 < x_2 \Rightarrow f(x_1) < f(x_2) \]

  • Decreasing on interval \(I\): \[ x_1 < x_2 \Rightarrow f(x_1) > f(x_2) \]

Example 1.8 Increasing and Decreasing Intervals

Let \(f(x) = x^3\).

Determine where the function is increasing or decreasing.

Solution

As \(x\) increases, the value of \(x^3\) always increases.

For any \(x_1 < x_2\): \[ f(x_1) = x_1^3 < x_2^3 = f(x_2) \]

Therefore, \(f(x) = x^3\) is:

  • Increasing on \((-\infty, \infty)\)
  • Decreasing on: none

1.1.5 Putting it All Together

Functions are the language of calculus:

They connect structure (math) to behavior (systems), data (tables), meaning (graphs), and rules (formulas).

1.1.5.1 Conceptual Takeaways

  • Functions model dependency.
  • Representations emphasize different insights.
  • Graphs reveal behavior.
  • Tables reveal data structure.
  • Formulas enable analysis.
  • Real-world functions often lack exact formulas but still support calculus operations.
  • Mathematical models approximate reality.

1.1.5.2 Skills You Should Be Able To Do:

  • Identify domain and range.
  • Convert between representations.
  • Interpret graphs.
  • Apply the vertical line test.
  • Construct piecewise functions.
  • Analyze symmetry.
  • Determine increasing/decreasing intervals.
  • Model real-world systems as functions.
